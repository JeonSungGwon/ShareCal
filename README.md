## 🖥️ 프로젝트 소개
일반적인 캘린더에서 나아가 일정을 공유하고 함께 정보를 기록해나갈 수 있는 일반 캘린더 + 공유 캘린더입니다.
<br>

### ⚙️ 개발 환경
- Spring boot
- Spring security
- JWT
- Spring Data JPA
- Docker
- AWS EC2, S3, Code Deploy, RDS(MySQL)
- Github actions
- Swagger
